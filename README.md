AUCC AXION FRAMEWORK
====================

## Installation and Usage

```bash
sudo apt-get install binwalk
sudo apt-get install exiftool
sudo pip install -r requirements.txt
git clone https://github.com/AUCyberClub/axion.git
cd axion
python Axion.py
```
