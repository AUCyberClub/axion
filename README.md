AUCC AXION FRAMEWORK
====================

## Installation and Usage

```bash
sudo apt-get install binwalk
sudo apt-get install exiftool
sudo apt-get install python-colorama
git clone https://github.com/AUCyberClub/Axion-Framework.git
cd Axion-Framework
python Axion.py
```
